My project for learning object orientated programming

Snake Class - with functions that creates the snake in different colors and spawns it on the screen, logic for controlling it, adds a segment when collision with "Food" class is detected, resets when collision is detected with itself or the border of screen

Food Class - Spawns randomly on the screen , will clear and spawn a new "food" of a random color on the screen if collision made with snake

Scoreboard Class - Starts with 0 points and increases by 1 each time "food" collected successfully and without losing the game. If game is over the current score will be stored in a text that will be kept in storage to show your highest score even if you restart the game
